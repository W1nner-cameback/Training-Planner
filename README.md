## Описание программы
*"Training Planner — это консольное приложение для для планирования тренировок с фильтрацией. Позволяет
добавлять записи о тренеровках (дата/тип/длительность) и фильтровать по типу и дате "*

## Требования для запуска
Что нужно установить на компьютер:
- Python 3.4 и выше
  
*запускать по ссылке: https://github.com/W1nner-cameback/Training-Planner

# Как пользоваться
* при запуске появляется меню, где :
Есть поля ввода (дата, тип тренеровки и длительность)
Есть кнопкадобавть тренеровку
Есть функция фильтрации (поля ввода с фильтрацией по типу и по дате, кнопки применить и сбросить фильтр)
Есть интерфейс с отображением тренеровок.


* Для использования необходимо ввести дату, тип, длительность тренеровки в отведенные для этого поля. после чего необходимо нажать кнопку "Добавить тренеровку", после чего тренеровка
* появится на экране
* Для фильтрации в поле ввода "Фильтрация по типу" необходимо ввести тип тренеровки, которую вы хотите увидеть и нажать кнопку "Применить фильтр". Анологично в поле ввода "Фильтреция по *
* дате" необходимо ввести дату тренеровки, которую хотите увидеть и так же нажать кнопку "Применить фильтр". Что бы сбросить фильтры и увидеть все тренеровки необходимо нажать кнопку *
* "Сбросить фильтр"
